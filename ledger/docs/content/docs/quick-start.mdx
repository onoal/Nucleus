# Quick Start

Build your first ledger in 5 minutes.

## Step 1: Create a Project

```bash
npx create-onoal-ledger
```

Follow the prompts to create your ledger project.

## Step 2: Install Dependencies

```bash
cd my-ledger
pnpm install
```

## Step 3: Start Development Server

```bash
pnpm dev
```

Your ledger is now running on `http://localhost:3000`.

## Step 4: Submit a Proof

```bash
curl -X POST http://localhost:3000/ledger/submit \
  -H "Content-Type: application/json" \
  -d '{
    "subject_oid": "oid:onoal:user:alice",
    "issuer_oid": "oid:onoal:org:identity",
    "target": "oid:onoal:asset:123",
    "type": "AgeOver18",
    "payload": {
      "age": 25,
      "verified": true
    }
  }'
```

## Step 5: Query Proofs

```bash
curl "http://localhost:3000/ledger/proofs?subject_oid=oid:onoal:user:alice"
```

## Step 6: Get Proof by ID

```bash
curl "http://localhost:3000/ledger/proof/{proof_id}"
```

## Using the Ledger Programmatically

Instead of curl, you can use the ledger directly:

```typescript
import { createLedger, coreRoutesModule } from "@onoal/ledger-core";
import { sqliteAdapter } from "@onoal/ledger-database-sqlite";
import { proofModule } from "@onoal/ledger-module-proof";

const ledger = createLedger({
  adapter: sqliteAdapter({ path: "./ledger.db" }),
  modules: [
    coreRoutesModule(), // Core routes (health, hash-root, verify, stats, etc.)
    proofModule(),
  ],
});

await ledger.initialize();

// Submit a proof
const entry = await ledger.append({
  type: "proof",
  issuer_oid: "oid:onoal:org:identity",
  subject_oid: "oid:onoal:user:alice",
  payload: {
    subject_oid: "oid:onoal:user:alice",
    issuer_oid: "oid:onoal:org:identity",
    target: "oid:onoal:asset:123",
    type: "AgeOver18",
    proof_data: {
      age: 25,
      verified: true,
    },
  },
  stream: "proofs",
});

// Query proofs
const { entries } = await ledger.query({
  stream: "proofs",
  subject_oid: "oid:onoal:user:alice",
});

// Get entry by ID
const proof = await ledger.get(entry.id);
```
